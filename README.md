**Enterprise-Grade SaaS Business Intelligence Tool**

AuraAnalytics is a powerful, modern BI dashboard designed to bridge the gap between raw data and actionable insights. This tool allows users to import large datasets, sync them with a MySQL repository, and generate professional visual reports instantly.

## üöÄ Key Features
* **Intelligent Data Pipeline:** Support for CSV and XLSX imports with automatic row analysis.
* **MySQL Integration:** One-click synchronization to move local data into a persistent cloud or local database.
* **Professional UI/UX:** Features a "Frosted Glass" navigation bar, blurred sticky headers, and responsive layouts.
* **Dynamic Pivot Workspace:** Build custom tables by selecting dimensions, measures, and aggregation types.
* **AI Insight Stream:** A built-in logic engine that highlights key patterns in your data.
* **Executive Reporting:** High-fidelity PDF export capability for board-room presentations.
